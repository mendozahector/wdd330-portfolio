JavaScript Grammar

Semicolons are not required, but are error-prone. Use following format instead:


        const message = 'Hello World!';alert(message);
        or
        {// this is a block containing 2 statements
          const message = 'Hello!';
          alert(message);
        }

Variables


        const name = 'Alexa'; // This won't be assigned to another string
        let score = 0; // This value may change during the program
        let x = 3, y = 4, z = 5;

Even if you use const to declare a variable, non-primitive data types can still be mutated later in the program. Using const prevents you from reassigning a variable to another object, as it will produce an error 


        const name = { value: 'Alexa' }; // an object
        name.value = 'Siri'; // change the value
        << 'Siri'

        // declare object
        const name = { value: 'Alexa' };

        // attempt to reassign to another object
        name = { value: 'Siri' }
        << TypeError: Assignment to constant variable.
